Army Income Calculator - User Guide

Overview:
 
The program is broken into 4 main areas (Selection, Employee, Allowance and Summary)
This program is designed to calculate both yearly and fortnight payments for Australian Army personnel in the Signals Corps, primarily Information Systems Technicians (ECN 661), Communication Radio Operators (ECN 662) and Telecommunications Technicians (ECN 665).

 
Rank Selection (Limited to 8 ranks) – Selection Area:
To select your rank, press on the button that corresponds with the rank you want to select. Once you select the rank the text field will show the rank that is currently selected. Only ONE rank can be selected.
SIG = Signaller (Straight out of training)
PTE(P) = Private Proficient (1 Year out of training)
LCPL = Lance Corporal
CPL = Corporal
CPL[1] = Corporal (1 Year in rank)
CPL[2] = Corporal (2 Years in rank)
SGT = Sergeant
SGT[1] = Sergeant (1 Year in rank)
 
ECN Selection (Limited to 3 trades) – Selection Area:
Select the appropriate ECN (Employee Categorization Number). Once you select an ECN the text field will show the currently selected ECN. Only ONE trade can be selected.
661 = Information Systems Technician
662 = Radio Systems Operator
665 = Telecommunications Technician
 
Special Forces Tier – Selection Area:
If calculating for a special force’s member select the appropriate tier, tier units are based off course completed for trades. J is a pay grade increase and K is 2 pay grade increases. If none are appropriate select “None”. Once you select a tier or “None” the text field will show the selected value. Only ONE Tier can be selected.
 
 
Allowances Selections (Limited to 5 Allowances) – Selection Area / Allowance Area:
Five of the main allowances have been added to this program, click on the appropriate ones and they will be added to the “Allowances Added” List. If you mis-select one, simply press “Clear Allowances” below the list as seen in the picture below to start over and re add allowances as necessary. Multiple allowances can be selected.
 
 
Living situation – Selection Area:
As all living situations directly effects the pay of military members there are only three options for living situation. Select as appropriate. The text field will show the currently selected option. Only ONE situation can be selected.
 
Superannuation – Selection Area:
Superannuation is set through a system called PMKEYS and can be set anywhere between 5%-10%.
Select the appropriate value and the text field will show the the current selection. Only ONE rate can be selected.
 
Before tax deductions – Selection Area:
If you want to add in any before tax deductions press the “Yes” button and type the dollar amount that is paid per fortnight. The text field will show only “Yes” or “No” for the current selection.
 
 
Summaries / Calculations – Summary Area:
The Yearly and Fortnight summaries are calculated using all the values selected in the above listed options, please ensure that all values have been selected even if it is “None” or “No”.
Press “Calculate” to calculate either yearly or fortnight values to display a summary of pay.
Press “Clear” to clear the summary text field but not the values selected.
Results will be show as seen below.
 
Employees – Employee Area
This is a list area to store all calculations, inspect items in the list, Save to a file and load from previous calculations.
To save to file, Simply press “Save to file” and it will save the values into a csv file located in the same directory as the program.
To load a previous file, Press “load from file” and search for the previous file to import into the list.
To inspect a employee in the list simply select the employee in the list and press inspect and the Summary area will populate the values for that employee.
