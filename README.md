

*****************************************************************************
   # ADF Pay Calculator
*****************************************************************************

 STILL IN DEVELOPMENT

*****************************************************************************

 Features :
  
   - Calculates Annual income with Allowances (Clothing / Seperation / SF etc).
   - Displays both Annual and Fortnight Figures.
   - Calculates Tax (Annually and Fortnightly).
   - Calculates your Super Contribution based of % you have set.
   - Displays both RA and DHA pay figures (Annually and Fortnightly).
   
*****************************************************************************

 How to use :
   1.  Download .jar
   2.  Run .jar file
   3.  Enter in values listed from your payslip
   4.  Eat popcorn and enjoy
